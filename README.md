O projeto consiste em simular os dados de venda para uma confeitaria, utilizando a linguagem Python . Foi utilizado como ferramenta para o projeto as bibliotecas Pandas, para a manipulação de dados, Numpy para algumas operações e Seaborn para a visualização.

Tecnologias empregadas:
- Python
- Pandas para a análise de dados
- Matplotlib/Seaborn para gráficos de faturamento

Diferenciais:
- Separação dos produtos por unidade (Kg e Cento)
- Análise do faturamento mensal e por categoria de produto
